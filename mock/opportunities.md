## 2. Code Bounty - Detailed Description

## Mission

Build a production-ready NFT minting dApp that showcases Metaplex's Bubblegum program for compressed NFTs. This bounty aims to create a reference implementation that developers can learn from and fork for their own projects.

## Scope & Requirements

### What You Need to Build

A complete web application that allows users to:

1. Connect their Solana wallet (Phantom, Solflare, Backpack)
2. Upload an image and metadata for an NFT
3. Mint compressed NFTs using Metaplex Bubblegum
4. View their minted NFTs in a gallery
5. Transfer NFTs to other addresses

### Technical Stack

**Required:**

- React 18+ or Next.js 14+
- TypeScript
- Metaplex Bubblegum SDK
- Solana Web3.js
- Wallet adapter

**Recommended:**

- TailwindCSS for styling
- shadcn/ui for components
- React Query for data fetching

### Code Quality Standards

Your submission must include:

- ‚úÖ Well-documented code with JSDoc comments
- ‚úÖ TypeScript with strict mode enabled
- ‚úÖ Error handling for all wallet interactions
- ‚úÖ Loading states and user feedback
- ‚úÖ Mobile-responsive design
- ‚úÖ README with setup instructions
- ‚úÖ At least 80% test coverage (unit tests)

## Deliverables

1. **GitHub Repository** - Public repo with clean commit history
2. **Live Demo** - Deployed on Vercel/Netlify (devnet)
3. **Documentation** - README + architecture overview
4. **Demo Video** - 2-3 minute walkthrough

## Evaluation Criteria

- **Code Quality** (40%) - Clean, maintainable, well-structured
- **User Experience** (30%) - Intuitive interface, smooth interactions
- **Innovation** (20%) - Creative features beyond basic requirements
- **Documentation** (10%) - Clear setup guide and code comments
